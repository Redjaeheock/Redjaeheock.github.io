# 인공지능 및 머신러닝 소개

## 머신러닝과 딥러닝이란 무엇인가

### \[ 머신 러닝 ]

- 데이터로부터 학습
- 우선 시도 후 **예상된 답과의 차이를 계속 줄여감**
	- 사람의 학습 방법과 유사
- 예시![[Pasted image 20250819071456.png]]
		
	- 분류/판별
		- 특징인자
			- **평균**
		- 심각판별
			- 심각의 경우 **노이즈 값** -> 평균으로 도출하기 어려움
			- **분산을 활용**해서 도출
				- 얼마만큼 평균과 떨어진지를 확인해 노이즈 값을 확인 가능
		- **Key Feature 선택이 중요!**
			
		- 분산에 따라 데이터가 분류되면 **그 경계를 지어주는 것이 모델 구축**(Build)![[Pasted image 20250819071957.png]]
		- 선은 직선, 타원, 로그 그래프 등 다양한 방식이 존재
- 머신러닝 알고리즘은 **라이브러리 형태로 제공**됨
	- 특징을 무엇으로 할 것인가를 설정하여 적정 라이브러리 사용
- 위의 예시처럼 평균 + 분산을 통해 정상, 주의, 심각 데이터를 구분하는 것을 **Feature Engineering**이라고 한다
- 응용 분야의 영역 지식 즉, **도메인이 매우 중요**
- 잘못된 데이터가 입력된다면 아무리 좋은 알고리즘이라도 소용 없음
	- *'Garbage in, Garbage out'* - 쓰레기 값이 들어가면 쓰레기 값이 나온다
	- **데이터 가공, 데이터 선택, 데이터 특징점 찾는 것이 매우 중요**

### \[ 머신러닝과 딥러닝]

#### 머신러닝
- 특징점 데이터를 사람이 제공
#### 딥러닝
- 특징점 데이터를 딥러닝이 스스로 찾아냄

![[Pasted image 20250819073159.png]]
#### 딥러닝이 무조건 좋은 것인가?
- 딥러닝은 특징점을 직접 찾아내기 위해 빅데이터를 요구
- **데이터가 부족**하다면 **머신 러닝**을 활용하는 것이 바람직
- **특징을 지정하기 어려운** 경우 **딥러닝**을 활용하는 것이 효율적
