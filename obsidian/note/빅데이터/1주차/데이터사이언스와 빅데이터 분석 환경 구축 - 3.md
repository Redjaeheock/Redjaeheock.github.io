	- ![[Pasted image 20250820232033.png]]

		- 디렉토리를 변경하면 하단 터미널은 사라진다.
			- 새로운 작업 영역으로 인식하기 때문
	- 해당 폴더 선택하면 이전에 저장해둔 파일(ex01_1.py)의 존재를 확인할 수 있다.

	- 파일 편집란에 다음과 같이 작성해보자

-  이전의 터미널 생성한 내용을 통해서 다시 터미널 생성
	- 터미널의 경로를 보면 이전과 다른 현재 작업 중인 경로가 설정돼 있는 것을 확인할 수 있다.
	- 터미널에 Python 이라고 입력한 후 키보드 \[ Tab ] 키를 누르면 현재 경로의 .py 코드를 순회할 수 있다.
	- \[ Tab ] 키로 ex01_2.py 실행
		- `hello world`
		  `hello world`

	- 터미널의 상단 메뉴바 빈 곳을 우클릭하면 다음과 같은 메뉴가 있다.
		- 오른쪽으로 이동
		- 왼쪽으로 이동
		- 패널 숨기기
	- 터미널에서 키보드 방향키 \[ ↑ ] 키를 누르면 이전에 실행했던 명령어들을 볼 수 있다.
		- 이전 명령어를 통해 ex01_2.py 를 ex01_01.py 로 수정해서 실행해보자
		- `빅데이터프로그래밍` 출력됨
	- VS Code 창의 우측 상단 근처의 재생 버튼을 클릭하면 현재 작업중인 파일이 바로 실행된다
- 파이썬 확장 설치하기
	- \[ Extensions ] 에서 Python 검색 후 Microsoft 의 Python 을 install 하기
		- 다운로드 수가 많은 패키지가 비교적 안전하고 안정적이다
	- Python 만 install 했지만, 패키지 내역을 보면 Jupyter 나 Pylance 등이 같이 설치된 것을 확인할 수 있다.
- 등간격 글꼴 다운받기
	- 메모장에 영어 대 $\cdot$ 소문자 입력시의 간격과 한국어의 한글 부분들이 간격이 같지 않는

	  좋은 편이다.
	- [D2Coding](https://github.com/naver/d2codingfont)로 이동
		- 레포지토리 목록에서 보이는 날짜로 최신 zip 을 클릭
		- Download 클릭
			- 다운받는 장소를 Tools 로 설정
		- 압축해제 및 해제된 폴더로 이동
		- D2CodingAll 폴더로 이동
			- 폴더내의 파일 클릭
			- \[ 설치 ] 탭 클릭
			- ![[Pasted image 20250821014154.png]]
	- VS Code 창의 \[ 파일 ]  -> \[ 설정 ] -> \[ 기본 설정 ] 클릭
		- 좌측의 \[ 텍스트 편집기 ] -> \[ 글꼴 ] 클릭
		- 화면의 글꼴 설정에서 Font Family 입력 칸에 D2Coding 으로 입력
		- 안 녕 하 세 요
		  AB CD EF GH IJ
		  ab cd ef gh ij
		  를 입력해보면 간격이 일정하게 정립되는 것을 볼 수 있다
		- 메모장에서도 사용 가능한 글꼴이다

### \[ Jupyter Notebook ]
#### Python3 를 설치하고, VS Code 에서 작업하는 것은 '파일 단위 실행' 과정
	
- 매번 코드를 작성 $\cdot$ 수정하고 저장한 후에 터미널이나 VS Code 의 Python runner 를 통해 
  파이썬 파일을 돌려야 결과 확인이 가능 
#### Jupyter Notebook 은 '셀(Cell) 단위 실행'으로 동작된다
	
- 셀 단위 실행이 가능하여 원하는 코드를 작성하고, 내가 원하는 부분(Cell)만을 실행하여 결과 확인이 가능
	- 부분적인 단위의 코드를 작성해서 그 단위, 단위 마다의 셀을 실행해서 결과를 확인한 수 있는 방법
- Visual Studio Code 에 통합되어 아나콘다(Anaconda)를 설치 없이도  Jupyter Notebook을
  사용할 수 있다.

#### Jupyter Notebook 실행하기
- VS Code 창에서 \[ 보기 ] 탭의 \[ 명령 팔레트 ] 클릭
- VS Code 창 가운데 생성된 명령 팔레트에 Jupyter 입력
	- 리스트 중에서 '**Jupyter: Create New Blank Notebook**' 클릭
	- Untitled-1.ipynb 파일 편집기가 열림
	- 보이는 첫 셀 에다가 다음과 같이 작성해보자
	  msg = "처음 사용하는 주피터노트북"
	  print(msg)
	- 셀의 앞부분에 있는 Run cell 버튼 클릭
		- Jupyter Notebook 을 VS Code 에서 처음 사용한다면 iPykernel 필요하다는 경고문 창이 뜬다
			- 창의 install 버튼을 틀릭
		- 설치가 완려되면 Run cell 버튼을 다시 클릭
			- 셀 바로 밑에
			  `처음 사용하는 주피터노트북`
			  출력됨
		- 다음 셀에 다음과 같이 입력
		  print("hello world")
		  print("hello world")
		  print("hello world")
		- 실행
		  `hello world`
		  `hello world`
		  `hello world`
		  이 셀 바로 밑에 출력됨
		- ![[Pasted image 20250821020855.png]]
	- 셀 실행시 Run cell 버튼이 아닌 \[ ctrl ] 키 + \[ Enter ] 를 입력해도 실행을 할 수 있다
	- Save notebook 클릭하면 파일 경로 설정하는 창이 열린다
		- 파일 명을 ex01_3 으로 저장
		- VS Code 의 좌측 파일 관리 파트를 보면 ex01_03.ipynb 확장자로 저장되는 것을
		  확인할 수 있다.
