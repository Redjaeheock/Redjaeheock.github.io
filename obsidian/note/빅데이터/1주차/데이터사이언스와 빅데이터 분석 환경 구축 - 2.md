# 빅데이터의 응용 분야와 파이썬

### \[ 빅데이터 응용 분야]

**유동 인구 행동 패턴**
- 서울시 시간대별 생활 인구의 유출입 정보
	- 유동 인구의 지역별 성별 연령에 따른 통행 패턴(통근, 통학, 주거, 여가 등)에 대한 경향 분석
	- 서울시는 2018 년 3월부터 서울 열린데이터광장에 서울 생활 인구 데이터 제공
	- 데이터 가공 $\cdot$ 수집 절차
		1.  통신 기록 수집
			- 전국 KT LTE 데이터 망 사용 휴대폰의 기지국 통신 데이터 수집
		2.  통신 기록 가공
			- KT 외 다른 통신사 사용자 등 집계 대상을 전국민으로 확장하기 위해 다른 
			  통계 자료와 비교 보정
		3. 통계 자료 배포
			- 보정한 데이터를 집계구 단위로 배분하여 통계 작성
			- 서울 열린데이터광장 통해 배포
-  중심업무지구 유동 인구 특성
	- 노동인구인 20 ~ 50대의 인구가 주요 노동 시간인 8시 ~ 20시 사이에 분포
	- 30 대 이후부터 남성이 더 많은 것을 확인 할 수 있음
		- 30 대 이후 여성이 결혼과 육아 등의 이유로 직장을 그만두는 현상의 영향으로 유추

	- 주 수요층인 20대 젊은 층을 대상으로 한 상점 $\cdot$ 유흥가가 새벽까지 이용하는 20대 인구가 많이 분포

**입지선정: 배달 권역 분석**(피자집)
- 음식점 점포 이전 시 피자 배달을 많이 하는 고객들에게 밀착서비스를 제공하기 위해 사례
	- 주거 지역에 피자 배달 위해 요구되는 고객
		- 내점 고객
		- 포장 고객
		- 배달 고객

**고객 유치: 신용카드**
- 멤버십 카드
	- 고객, 매출, 주소 텍스트 데이터 등을 이용해 점포 별 내점, 포장, 배달 데이터를 구분 관리
		- 예시
			- 점포매출: 내점 30%, 포장 10%, 배달 60% 구성
	-  배달고객 주소 정보로 '지오코딩(Geo-coding)'을 이용
		- 지번과 위치 정보를 연결시켜 위 $\cdot$ 경도 좌표 값으로 변환
		- 컴퓨터 지도 뒤에 위 $\cdot$ 경도의 좌표 값을 표시하면 고객 위치 정보가 표시
- 신용카드 빅데이터
	- 신용카드 빅데이터 이용하여 고객, 업종, 매출, 상권에 대한 객관적인 데이터 참고 가능
	- 신용카드의 자택과 직장 주소 정보로 고객에 관한 관심지역의 주거민, 직장인, 유동객 
	  비율 파악 가능
	- 성별 연령대를 통해 청녕, 장년, 노년층의 비율 파알하여 어떤 고객에게 집중할 것인지
	  선택 가능
	- 관심 지역에서 주로 소비하는 고객층의 소득 수준에 따라 비싼 상품을 팔지, 저가 가격대 유지할지 고려 가능
-  신용카드 매출 분석
	- 신용카드의 매출이 시간 단위로 분석될 때 소비 특징과 흐름을 파악 가능
	- 하루 중 언제 가장 소비가 활발하고 줄어드는지 파악 가능
	- 요일 별로 살펴보면 주중과 주말 어떻게 매출이 다른지 파알 가능
	- 직장인이 많은 지역은 주말에 한가, 반면 주말에만 붐비는 곳도 존재
	- 월 단위와 계절 단위 소비 특징
		- 해수욕장 VS 스키장
	-  데이터가 쌓이면 최근 몇 년 동안 관심 지역의 매출 변화 파 악 가능
- 프랜차이즈 점포 소비 패턴 분석
	- 음식점에서 카드 결제한 데이터로 식당 별로 고객의 특징 파악 가능
	- 카드 사용자의 성별, 연결, 주거지, 생활 특성, 시간대 별 특징 파악
	- 추정 소득도 따로 계산하여 소득 수준 별 소비 특징 분석 가능
- 신용카드 빅데이터는 점주나 예비 창업자에게 **장사 현황**, **소비 형태 특징**, **경쟁 상황**, **유행 변화** 등을 객관적인 통계로 제공 가능

### \[ 파이썬 개념 ]

**파이썬**
- 귀도 반 로섬(Guido van Rossum)이 1991년 개발
- 초보자가 쉽게 배울 수 있는 프로그래밍 언어
- R 과 함께 빅데이터 분석 언어로 각광 받음
- 장점
	- 비전공자도 쉽게 비울 수 있음
	- 다양한 분야에서 활용할 수 있음
	- 대부분이 운영체제에서 동일하게 사용됨
- 단점
	- C 언어에 비해 일반적으로 10 ~ 350배 느림
	- 최근 컴퓨터 성능이 좋아져서 연산이 많이 필요한 프로그램이 아니라면 큰 차이를 느낄 수 없음

**파이썬 설치하기**
1. 파이썬 설치 프로그램 다운 [파이썬](http://www.python.org)
	1. 파이썬 홈페이지 이동
	2. \[ Downloads ] 탭에서 OS 에 맞춰 다운받기
2. 로컬의 C 또는 D 드라이브에 폴더명 'Tools'로 폴더 생성 
	- 다운받은 파이썬 설치 파일 복사(잘라내기)해서 붙혀넣기
3. 파이썬 설치 파일 실행
	- 설치 창이 열리면 창의 하단에 '**Add Python 3.x PATH**' **박스란 체크**하기
		- 환경 변수 자동 설정 
		- 체크없이 설치할 경우 실행이 안됨
			- 별도로 환경 변수 설정해줘야 됨
4. 설치 창의 다음 화면의 Install Now 클릭
5. 설치 과정 마지막에 close 버튼이 나오면 클릭
6. 시작 메뉴를 통해 Python 3.x 디렉토리 존재 확인

**파이썬 인터렉티브 셀**
- 파이썬 명령어를 한 줄씩 입력하며
- 프롬프트(Prompt)
	- 꺽쇠 3개(>>>) 로 이루어져 코드를 한 줄씩 입력
- 대화형 셀
	- 컴퓨터와 상호 작용하는 공간, 한 마니씩 주고 받는 것처럼 대화한다고 하여 대화형 셀이라고도 불림

**파이썬 다뤄보기**
-   Python 3.x 디렉토리에서 **Python 3.x 실행 프로그램 실행**하기
	- 프롬프트 창 활성화 됨
-  활성화 된 프롬프트 창에 여러가지 테스트 해보기
	- 10 + 10
		- `20` 출력됨
	- "10" \* 10
		- `'10101010101010101010'` 출력됨
		- "10" 를 숫자가 아닌 문자로 보고 문자를 \*10 번 출력
	- print("Hello")
		- Hello 출력됨
-  Python 3.9 디렉토리에서 Pytho **IDLE Shell 3.x 실행**
	- 활성화 된 shell 창에 여러가지 테스트 해보기
		- 10 + 10
			- `20` 출력됨
		- print("hello")
			- `hello` 출력됨
	-  shell 창의 \[ File ]탭에서 \[ New File ] 클릭
		-  코드 편집기 창이 생성됨
		- 생성된 편집기에 간단한 코드 작성해서 실행해보기
			- print("빅데이터프로그래밍")
				- 문자를 입력할 때 더블쿼터(겹따옴표)를 항성 넣어줘야 한다
		- 편집기 창의 \[ File ] 탭에서 \[ Save ] 클릭
			- 저장 경로 설정 창 활성화 됨
			-  원하는 경로에 파일들을 저장할 디렉토리 생성하기
				- 저장할 파일 명(ex01_1.py) 설정해서 저장
					- 저장하면 편집기 상단의 \*untitled* 부분이 저장한
					  파일 명으로 바뀜
		- 저장 후 편집기 창의 \[ Run ] 탭에서 \[ Run Module ] 클릭
			- shell 창에 `빅데이터프로그래밍` 출력됨
