
## 서로소

#### 예시 1: N=3N=3N=3

전체 집합
$$D=\{(i,j)∣1\leq i,\:j\leq3,  j≠i\}$$

원소를 쓰면:
$$D=\{(1,2),(1,3),(2,1),(2,3),(3,1),(3,2)\}$$


이제 둘로 나눕니다:

- $D_1 = \{(i,j)\in D \mid i<j\} = \{(1,2),(1,3),(2,3)\}$
    
- $D_2 = \{(i,j)\in D \mid i>j\} = \{(2,1),(3,1),(3,2)\}$


여기서 보면 $D_1\;$​의 각 원소에 대응되는 **짝**이 $D_2\;$​에 존재합니다.  

예: $(1,2)\leftrightarrow(2,1),\; (1,3)\leftrightarrow(3,1),\; (2,3)\leftrightarrow(3,2)$.

따라서

$$\sum_{(i,j)\in D} A_{ij} = \sum_{(i,j)\in D_1}(A_{ij}+A_{ji})$$

## 예시 2: N=4N=4N=4

전체 집합

$$D=\{(i,j)∣1\leq i,\:j\leq4,  j≠i\}$$

원소는 총 $4×3=12$

$$D = \{(1,2),(1,3),(1,4),\;(2,1),(2,3),(2,4),\;(3,1),(3,2),(3,4),\;(4,1),(4,2),(4,3)\}$$

분할하면:

- $D_1=\{(i,j)\in D\: |\: i<j\} = \{(1,2),(1,3),(1,4),(2,3),(2,4),(3,4)\}$
    
- $D_2=\{(i,j)\in D \mid i>j\} = \{(2,1),(3,1),(4,1),(3,2),(4,2),(4,3)\}$
 

마찬가지로 각 원소는 1:1 대응:  

$(1,2)\leftrightarrow(2,1),\; (1,3)\leftrightarrow(3,1),\; …,\; (3,4)\leftrightarrow(4,3)$

$$\sum_{(i,j)\in D} A_{ij} = \sum_{(i,j)\in D_1}(A_{ij}+A_{ji})$$
## 요약

- **원래 조건** $j\neq i\;$는 $D\;$라는 집합을 정의. (자기 자신 제외한 모든 순서쌍)
- 이것을 $i<j\; 와\; i>j$ 두 부분집합 $D_1,D_2$로 나누면 정확히 절반씩.
- 따라서 합을 쓸 때는
#### $$\sum_{(i,j)\in D} A_{ij} = \sum_{(i,j)\in D_1}(A_{ij}+A_{ji})$$
