## OT

단어, 이디엄(idiom) 숙지

1. 첫 try 는 가볍게
2. return 시 세심하게
	- 보편적인 문제와 실력을 물을 수 있는 문제를 출제 섞어서
	- 인지가 약한 부분을 잘 마킹하고 체킹
		- 집중적 회독 수 늘리기
3. 확실하게


어법, 작문
- 어법 － 부정사, 동명사, 분사를 잘 적용하는 원리의 기술
- 어법들이 뭉쳐서 to - to부정사, so-that 구문, the last whe(?) to 부정사 구문 등을 만들어 냄
- **중요도에 따라서** 마킹
	- 마킹의 정도에 따라 반복 주기도 많이

리딩, 회화
- 리딩과 회화가 자연스러워야만 글로벌적 적으로 학문적 소양을 갖췄다고 말할 수 있음
- 리딩
	- 어휘, 이디엄, 어법 등이 하나의 어우러짐으로써 글 속에 있는 참 뜻을 읽어내기
- 테마에 맞게 꾸준히 연습
- 자신의 약한 부분을 체계적이고 집중적으로 하기
