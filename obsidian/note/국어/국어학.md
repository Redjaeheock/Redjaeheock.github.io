## 국어에 대한 이해

### < 언어로서의 국어 >

#### \[ 언어의 체계와 사고 ]

언어 기호는 각각 개별적으로 존재하는 것이 아니라 다른 언어 기호들과 함께 공존하면서 
상호 관계를 가지고 그 가치를 발휘
언어 기호는 말소리, 어휘 그리고 문법 규칙 같은 하위 체계들로 이루어짐

- ##### 언어의 체계
	- **언어 체계의 성격**
		- **계층성**
			- 하위의 체계는 또 다른 하위의 체계로 이루어진다
		- **긴밀성**
			- **체계를 이루는 각 항목**이나 **범주**는 서로 긴밀히 **연관되어** 있기 때문에 
			  그 중의 하나가 변화를 입게 되면 체계 전체가 변화한다
	- 언어 체계와 선택
		- **체계**
			 - 서로 긴밀한 관련을 가지고 있는 선택 가능항의 집합
		- **선택**
			- 언어 사용자가 한 체계를 구성하는 항목들 중에서 필요한 것을 골라 쓰는 것
			- **폐쇄적 선택**
				- 선택 **가능항의 수가 제한**적인 경우
					- **문법** 범주
				- 어떤 말 다음에 올 수 있는 한정적인 것들
				- 우리 국어는 SOV 형식으로 'v' 서술어로써 용언이 올 수 있다
			- **개방적 선택**
				- 선택 **가능항의 수가 거의 제한이 없는** 경우
					- **어휘** 범주
- ##### 언어와 사고
	- 언어와 인간의 사고는 매우 밀접한 관계
	- 언어가 사고에 우선하는지, 사고가 언어에 우선하는지에 대해서는 관점의 차이가 존재
	- **언어 우위론**
		- 사고보다 **언어가 먼저**라는 견해
		- 언어 없이는 사고가 불가능하다는 관점
		- 언어로 명명해야만 대상으로 인식할 수 있다고 본다
	-  **사고우위론**
		- 언어보다 **사고가 먼저**라는 견해
		- 언어 없이도 사고는 가능하다는 관점
		- 명명 과정 없이도 존재할 수 있다고 본다

#### \[ 언어의 특성 ]

- ##### 기호성(記號性)
	- ( 記 기록할 **기**, 號 부르짖을 **호**, 性 성품 **성** )
	- 언어는 기호의 하나로, **형식**(말고리)과 **내용**(의미)을 가지고 있다
	
- ##### 자의성(恣意性) = 임의성(任意性) = 무연성(無緣性)
	- ( 恣 방자할 **자**, 意 뜻 **의**, 性 성품 **성** )
	- ( 任 맡길 **임**,  意 뜻 **의**, 性 성품 **성** )
	- ( 無 없을 **무**, 緣 인연 **연**, 性 성품 **성** )
	- **말소리와 의미 사이**는 **아무런 필연적인 관계가 없는 임의적 관계**
	- 모든 언어가 어떤 의미를 전하기 위해 한 가지로 정해진 말소리를 사용하는 것은 아닌 각각의 말소리들은 우연히 그렇게 결정된 것일 뿐이다
	- 예시
		- 한국어 - 책상
		- 영어 - Desk
		- 같은 것을 다른 형식으로 표기
	
- ##### 사회성(社會性) = 불역성(不易性)
	- ( 社 모일 **사**, 會 모을 **회**, 性 성품 **성** )
	- ( 不 아닐 불, 易 바꿀 **역**, 性 성품 **성** )
	- 언어는 **사회적 약속**이므로 **개인이 함부로 바꾸지 못한다** 
		- 소쉬르 曰
	- 언어는 한 사회 공동체가 공유하는 것이므로, 그 사회에서의 묵계가 성립되지 않는 
	  변개는 결과적으로 인간의 의사소통을 위한 표현 전달의 도구로서의 기능을 언어로부터 박탈하는 것이 됨
	
- ##### 역사성(歷史性) = 가역성(可逆性)
	- ( 歷 지낼 **역**, 史 역사 **사** , 性 성품 **성** )
	- ( 可 옳을 **가**, 逆 거스를 **역**, 性 성품 **성** )
	- 언어는 항상 고정되어 불변하는 것이 아니라 시**간의 경과에 따라** 단어의 **소리와 의미**가 
	  **변하거나, 문법 요소**에 **변화**가 생기면서 끊임없이 변화한다
	- 즉, 새말이 생기기도 하고, 있던 말의 형태나 의미가 변하기도 하고, 쓰이지 않아 
	  없어지기도 한다
	- 변할 수 있는 성질이라 하여 '가역성' 이라고 한다
	- 언어는 계속해서 **신생, 성장, 사멸**한다.
	
- ##### 분절성(分節性) = 불연속성(不連續性)
	- 현실 세계는 연속적이나 언어는 물리적으로 **연속된 실체를 분절적으로 끊어서 표현**한다
	-  **기호의 분절**
		- **단어와 어질**, **문장**은 소리의 **연속**이지만 **자음과 모음으로 나눌 수 있다**
	- **개념의 분절**
		- 언어는 외부 세계를 있는 그대로 반영하는 것이 아니라 **연속적을 이루어져 있는 세계**를 **불연속적인 것으로 끊어서 **표현
		- **연속적인 사물을 불연속적인 것으로 나누어 표현**
			- 시간, 무지개 등등
	
- ##### 추상성(抽象性) = 일반화
	- ( 抽 뽑을 **추**, 象 코끼리(, 형상) **상**, 性 성품 **성** )
	- 개념(언어의 의미)은 언어에 의해 분절이 이루어져 형성된 한 덩어리의 생각으로서,
	  **같은 부류의 사물들**에서 **공통적**으로 **속성**을 **뽑아내는 추상화의 과정**을 **거쳐서 형성**된다.
	- 언어는 본질적으루 추상적이다
	- 예시
		- 어류: 고래, 상어, 가자미 등등
		- 어류 - 추상화
		- 고래, 상어, 가자미 - 구체화
	
- ##### 개방성(開放性) = 창조성(創造性)
	- ( 開 열 **개**, 放 놓을 **방**, 性 성품 **성** )
	- ( 創 비로소 **창**, 造 지을 **조**, 性 성품 **성** )
	- 언어는 무한에 가까운 생각들을 표현하고 전달할 수 있는 개방적 기호 체계
		- 인간은 길이와 수에 제한 없이 무**한한 문장을 만들어 낼 수 있으며**, **듣는 사람 역시 그 의미를 금방 이해**할 수 있다
	- 언어로 말미암아 인간의 사고가 미치는 범위에 제한이 사라지게 되었다
		- 언어는 개방적이고 무한한 체계이기 때문에 
		  우리는 실제로 보았거나 존재하는 것 외에 , 
		  실재하지 않는 상상의 산물이나 관념적이고 추상적 개념까지 
		  **거의 무한하게 표현할 수 있다**
	
- ##### 체계성(體係性) = 유기성(有機性)
	- ( 體 몸 **체**, 係 이을 **계**, 性 성품 **성** )
	- ( (有 있을 **유**, 機 베틀 **기**, 機 성품 **성** )
	- 언어 기호가 모여서 일정한 의미를 전달할 때, **언어 기호들은 하나의 체계를 이루고**,
	  **일정한 규칙에 따라 배열**되며, **일정한 질서 아래 실현**된다
	- 음운 체계, 어휘 체계, 문법 체계 등이 모두 언어의 체계적 단위
	- **구조의 관점의 체계**
		- 형태소 -> 단어 -> 어절 -> 문장 -> 담화	
	  **음성과 관련한 체계**
		- 음운 -> 음절
- ##### 선조성(線條性)
	- ( 線 줄 **선**, 條 곁가지 **조**, 性 성품 **성**)
	- 둘 이상의 기호가 전후 위치 관계를 유지하면서 순차적으로 배열
	- 언어 기호가 시간적·공간적으로 선을 따라 차례차례 배열된다는 성질
	- 예시
		- 말할 때:
		    - “나는 학교에 간다”라는 문장은 /나/ → /는/ → /학/ → /교/… 이렇게 소리가 
		      **시간 순서대로** 이어져야 한다
		    - 동시에 여러 단어를 겹쳐 말하면 알아들을 수가 없다.
        - 글로 쓸 때:
	        - “사과”라는 단어도 **사 → 과**라는 순서를 지켜야 의미가 유지된다.
            - “과사”로 쓰면 의미가 달라져 버린다.

#### \[ 언어의 기능]

- ##### 표현의 기능
	- 화자의 **심리 내용**(감정이나 태도)을 **표현**하는 기능
	- 언어의 개념적 의미 뿐만 아니라 지시 대상에 대한 화자의 감정이나 태도를 나타낸다
	
- ##### 감화적 기능(=지령적 기능, 환기적 기능, 명령적 기능)
	- 미래의 특정 행위나 사건이 일어나게 하거나 일어나지 않게 하며, 그 성격이나 방향을 
	  조정하는 기능으로, 듣는 사람으로 하여금 **특정 행동을 하도록 하는 기능**
	- 말하는 사람의 마음을 표현한다는 점에서 표현의 기능과 다르지 않으나, 듣는 사람에게 감화 작용을 하여 실제 행동으로 옮기도록 한다는 점에서 표현의 기능과 차이가 있다
	- 감화적 기능의 **가장 직접적인 형**태는 **명령형의 문장**인데, 
	  이 외에도 **청유형, 의문형, 평서형 등**의 문장도 쓸 수 있다
	- 표어, 광고문, 선거 유세, 격언, 교통 표지 등은 주로 감화적 기능을 한다
	- **언어를 전달 받은 사람에게 내가 원하는 행동을 언어를 통하여 직접적 혹은 간접적으로  전달하여 행동을 행하겠끔 유도하거나, 언어를 전달 받은 이의 공감대를 자극함으로써 수행적인 일을 일으키는 것**
	
- ##### 친교적 기능(사교적 기능)
	- 말하는 사람과 듣는 사람의 **친교 확보** 및 **친밀한 관계를 확인하는 기능**
	- 이는 원만한 사회생활을 유지하는데 윤활유 같은 기능으로 인사말들이 대표적
	- 언어 기호의 **관습적**(개념적) **의미는 중시하지 않고**, **발화**(언어 $\cdot$ 행위) **자체를 중시**
		- ex) 동백 꽃 필 무렵
			- 점순이가 일하고 있는 주인공에게 혼자 일하냐고 묻는 장면에서 이는 문장 그대로 혼자 일하는 것을 묻는 게 아닌 인삿말 치레 전달 된 것
- ##### 표출적 기능
	- 언어를 기호 이전의 용법으로 **거의 본능적으로 사용되는 것**으로, 표현 의도나 전달 의도가 없는 것
	- 놀라거나 위험할 때, 현장에 듣는 사람이 있거나 없거나 관계없는 **무의식적으로 나오는 소리**
	
- ##### 지식과 보존의 기능
	- **지식을 보존하고 축적**해 가는 기능으로 **언어의 전달 기능과도 밀접하게 관련**되어 있다.
	- **과거에는 오직 문자 표기만**으로 지식 정보를 저장해서, 공간과 시간을 초월하여 
	  다른 사람에게 전달할 수 있었지만, **현대에 들어와서는 음성을 보존**하는 일도 가능해졌다
	
- ##### 미적 기능
	- 언어 예술 작품(문학 작품, 특히 시)에 주로 사용되는 것으로 **언어의 미적 가치**를 
	  추구하는 기능
	- 이 경우 감정적 의미만이 아니라, 언어의 형식인 **음성이 주는 효과**(개념적 의미)기 
	  **매우 중시**된다
	
- ##### 관어적 기능(사전적 기능)
	- 언어 수행에 필요한 매체로서의 언어의 기능에 충실한 기능으로, 말을 통해 새로운 말을 학습하고 지식을 증진시키는 역할을 한다
	- 메타데이터와 유사

### < 국어의 언어적 특징 >

#### \[ 국어의 본질 ]

- ##### 국어의 개념
	- 국어는 언어이다 
		- **국어의 일반성**
	- 국어는 일종의 구체적 $\cdot$ 개별적 언어이다
		- **국어의 특수성**
	- 국어는 국가를 배경으로 한다
	- 국어는 **정치상 공용어**(공식어), **교욱상 표준어**이다
	- 한 나라에서는 하나의 국어가 사용되는 것이 원칙이다. 
	  그러나 예외적으로 둘 이상의 국어를 사용하는 나라도 있다
	
- ##### 국어의 분류
	- **계통상 분류**
		- 국어는 알타이 어족에 속하는데, 알타이 어족에는 만주어(퉁구스어), 몽골어, 터키어, 한국어, 일본어 등이 있다.
	- **형태상 분류**
		- 국어는 첨가어(교착어 $\cdot$ 부착어)에 속함
		- 예시
			- 하라는 학원에 갔다
				- '는', '에' 조사
				- 갔다 -> 가-'았'-다
			- **교착어**(= 부착어)
				- 어근에 접사가 질서 있게 붙어 문법적 의미(시제·격·인칭 등)를 
				  표현하는 언어
				- 조사는 하나로써의 기능은 없지만, 단어와 붙으면서 의미를 발하게 된다
				- 예시 문장의 '는', '에' 가 해당
			- **첨가어**
				- 예문 '갔다' 는 '가다' 에서 선어말어미인 '았'이 붙으므로써 과거 시제를 
				  나타내줌
		- 첨가어는 문법적 기능을 담당하는 형식 형태소(**조사, 어미, 접사**)와 의미를 담당하는 실질 형태소가 분명이 구별되는 언어
		

| 분류               | 성격                                                                                                                                                                                                                                                            | 예                  |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------ |
| 첨가어<br>(교착어,부착어) | 뜻을 나타내는 성질 형태소에 조사와 의미, 접시 같은 문법적 관계를 나타내는 형식<br>형태소가 붙음으로써 문법적 기능을 다하는 언어<br>(체언+조사, 어간+어미, 어근+접사)를 말한다<br><br>ex) ***나 + 가(주격), 나 + 의(소유격), 나 + 를(목적격)***                                                                                                   | 한국어, 일본어, 몽골어, 터키어 |
| 굴절어              | 낱말에 다른 말을 첨가하지 않고, **어형(**語形)**의 일부를 변화**시키거나 또는 **접사를 붙여**, 단어가 문장 속에서 가지는 여러 가지 관계를 나타내는 언어<br>명사$\cdot$대명사 등에는 성(性)$\cdot$수(數)$\cdot$격(格),<br>동사에는 인칭(人稱)$\cdot$시제(時制)$\cdot$수(數) 등의 문법 범주가 있어 이에 따라 일정하게 변화한다<br><br>ex) ***I (주격), my (소유격), me (목적격)*** | 영어, 독어, 불어         |
| 고립어              | 어미의 변화나 접사 등 문법적 관계를 나타내는 요소의 발달이 없고, **낱말의 실현 위치에 의하여 단어가 문장 속에서 가지는 여러 가지 관계가 결정**되는 단어<br><br>ex) ***高山(높은 산) - 山高(산이 높다)***<br>                                                                                                                           | 중국어, 태국어, 베트남어     |
| 포합어              | 동사를 중심으로 그 전후에 인칭을 나타내는 접사나 목적을 나타내는 말이 결합 또는 삽입되어서, 한 말로써 한 문장을 나타낼 수 있는 언어를 말함                                                                                                                                                                              | 아메리칸 인디언의 말, 아이누말  |

		- **표음 문자**(소리글자)
			- **단음문자** - 자음, 모음으로 음절을 나눔(한글, 영어 등)
			  -> 음운 문자, 음소 문자
			- **음절 문자**

#### \[ 국어의 특질 ]

- **음운상**의 특징
	- **음운** - 음소 + 운소
		- 음소 = 자음, 모음
			- **분절음운**
		- 운소 = 억양, 장단
			- **비분절음운**
	- 국어 자엄 중 **파열음 계열**(파열음, 파찰음)은 예사소리, 된소리, 거센소리의 
	  세 갈래의 대립을 통해서 **삼중 체계**를 이룬다
		- **삼지적 상관속**
		- 참고
			- **영어, 독어, 불어 등**과 같은 인도-유럽 계통의 언어는 파열음 계열에서 
			  **안울림소리**(무성음)와 **울림소리**(유성음)의 **이중 체계**로 되어 있다.
	- 다른 언어에 비해 **마찰음**(ㅅ, ㅆ, ㅎ)**이 적다**
	  -> 영어에는 ' f, v, $\theta$ ' 등의 마찰음이 더 있다.
	- **두음 법칙**이 있다
		- 녀자 -> 여자
		- 랭동 -> 냉동
	- 음운상 **설전음**(r)과 **설측음**(l) 의 **구별이 불분명**하다
	- **음절 끝 위치**에 오는 **파열음들이 닫힌 상태**로 발음된다
		- **음절 끝소리 규칙**
			- ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ (가느다란 물방울)
	- 단모음의 수는 10개 (ㅏ, ㅓ, ㅗ, ㅜ, ㅡ, ㅣ, ㅐ, ㅔ, ㅚ, ㅟ) 나 될 정도로 많다
		- 이중모음은 11개
	- 모음조화 현상이 있다
		- 양성모음은 양성모음끼리, 음성모음은 음성모음끼리 합성되는 것
		- 중세 국어 시기엔 모듬조화를 철저히 지켰음
			- 훈민정음 창세시기
		- 현대 국어에선 음성모음의 소리값이 강해지면서 모음조화 현상이 파괴된 단어들이 표준어로 규정이 되는 경우도 생김
	- **모음동화**, ***자음 동화*** 현상이 있다
		- 비슷한 자음 또는 모음이 만나서 소리가 변하는 것
	- **음상의 차이와 소리의 길이**로 인하여 **어감이 달라지거나 의미가 분화**되는 경우가 있다
- **어휘(단어)상**의 특징
	- 국어의 어휘
		- **삼중 체계**
			- 고유어, 한자어, 외래어
	- 다량의 **한자어 유입과 구미어의 유입**으로 **차용어가 많다**
		- 특히, 한자어의 비중이 높다
	- 풍류를 즐기는 낙천적 $\cdot$ 정서적 $\cdot$ 감각적인 민족의 특성이 어휘에 반영된 결과로 
	  **감각어가 발달**하였다
	- **음성상징어가 발달**
		- **의성어**
			- 소리를 나타내는 말
		- **의태어**
			- 형태를 나타내는 말
	- **친족 관계를 나타내는 어휘가 발달**
	- **단어**에 **성(性)의 구분이 없고**, **명사**에 **수(數)의 개념도 없다**
	- **형용사**에 **비교급과 최상급이 없다**
	- 관사, 관계대명사, 의문대명사, 전치사, 접속사가 없다
		- 국어는 후치사이기 때문에, 같은 역할을 부사의 범주에 넣어서 사용
- **구문상**의 특징
	- 첨가어로 조사와 어미, 접사가 발달되어 있어서 대부분의 문법 기능이 이들에 의해 실현
	- **단어 형성법이 발달**
		- **파생법**
			- 어근 + 접사
		- **합성법**
			- 어근 + 어근
	- '주어 + 목적어 + 서술어' 의 구조
		- SOV 형의 언어
		- **미괄형의 문단** 구성
	- 어순의 자리바꿈이 비교적 자유롭다
	- **수식어는 피수식어의 앞에 온다**(= **좌분지 언어**)
		- 수식어(꾸미는 말)
			- 관형어
			- 부사어
		- 피수식어
			- 체언
			- 용언
	- **문장의 요소가 생략되는 일이 많다**
		- 주어 및 조사 생략이 많다
		- 단, 어미는 생략될 수 없음
	- 상하 관계가 중시되던 사회 구조의 영향으로 **높임말과 높임 표현이 발달**
	- **시제의 표시가 불분명**
	- **능동과 피동**, **주동과 사동의 개념**이 **막연한 점**이 있다
	- 단어의 **단수 $\cdot$ 복수 개념이 엄격하지 않다**
