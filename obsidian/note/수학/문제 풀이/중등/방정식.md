$x$ 에 대한 방적식 $3kx - 2b = ak + 4x - 4$ 가 $k$ 의 값에 관계 없이 항상 $x = 2$ 를 해고 가질 때, $a + b$ 의 값을 구하시오 (단, $a,\; b,\; k$ 는 상수이다.)

풀이1:
$$3kx - 2b = ak + 4x -4,\quad x = 2$$
$$3k\cdot x - 2b = ak + 4\cdot x - 4,\quad x=2$$
$$3k\cdot 2 - 2b = ak + 4\cdot 2 - 4$$
$$6k - 2b = ak + 8 - 4$$
$$6k - 2b = ak + 4$$

' $k$ 의 값에 관계 없이 항상 $x=2$'
= '$k$ 에 대한 일차식' 
= '$k$ 를 어떤 수로 넣어도 좌변과 우변이 같다는 뜻' 
= '$k$ 에 대한 항등식'

$$6k - ak - 2b - 4 = 0,\quad \forall k$$
$$(6-a)k + (-2b - 4) = 0,\quad \forall k$$

이 식이 성립되려면  $k$ 의 계수도 0 이고, 상수항도 0 이어야 한다

$$\begin{cases} 6 - a = 0 \\
 -2b - 4 = 0 \end{cases}$$
$$ 6 - a = 0,\quad a = 6 $$
$$-2b - 4 = 0,\quad -2b = 4,\quad -2b\cdot -\frac{1}{2} = 4\cdot -\frac{1}{2},\quad b = -2$$
답:
$$a + b = 6 + (-2) = 4$$
