##### Teachable Machine 실습하기

마스크를 쓴 것과 쓰지 않은 이미지 분류기 만들어보기기

- 홈페이지 상단 우측에 Get start(시작하기) 클릭
	- ![[Pasted image 20250827173102.png]]
	
- 클릭 후 페이지 화면
	- ![[Pasted image 20250827173140.png]]
	- 이미지, 소리, 포즈 데이터를 학습할 수 있는 페이지를 볼 수 있다.
	
- Image Project(이미지 프로젝트) 클릭
	- ![[Pasted image 20250827174914.png]]
	- 모델 아키텍처를 볼 수 있음
	- Class 1 - 일반 얼굴 사진
	- Class 2 - 마스크 착용한 사진
	
- Class 1 의 데이터를 사용하기 위해 촬영하기
	- 영상을 통해 프레임을 추출해서 여러 이미지로 생성
	- 마스크 없이 카메라를 통해 촬영
	- 조금씩 움직여서 다양한 사진이 찍히겠끔 유도
- Class 2 의 데이터 또한 촬영을 통해서 데이터 준비
	- 마스크 끼고 촬영하기
	- 조금씩 움직여서 다양한 사진이 찍히겠끔 유도
- 두 촬영으로 데이터 준비가 끝나면 Model Training(모델 학습시키기) 클릭
	- 구글 클라우드 서버를 통해 학습처리
	- 데이터 양에 따라 학습시간이 달라짐
	- **학습 하는 동안에 다른 화면으로 전환하거나 탭을 누르면 학습이 중단됨**
- Previes(미리보기)를 통해 테스트
	- 캠을 통해서 마스크 착용 여부에 따라 하단의 아웃풋이 달라지는 것을 볼 수 있음

		- 여러 사람이 사용할 경우 해당 사람들의 목소리도 같이 학습시키면 정확도가 올라간다

### < 포즈를 학습시키는 머신러닝 서비스 만들기 >

#### \[ 포즈를 학습시키는 머신러닝 서비스 개발하기 ]

- TM 접속
- 포즈(Pose) 프로젝트 클릭
- O, X 동작 인식을 위한 클래스명 설정
	- ![[Pasted image 20250829094029.png]]
	
- Class 별 학습 시 카메라를 통해 관련 동작을 취한다
	- 조금씩 움직여서 다양한 사진이 찍히겠끔 유도
	- 찍힌 사진 중에 동작과 관련 없는 사진이 있다면 개별 삭제도 가능
		- 삭제 과정에서 클래스 간의 데이터 수가 1 ~ 2 개 정도를 넘는 차이가 나지 않겠끔 주의해준다
- Training 클릭
	- 구글 서버를 통해 학습
	- 데이터 양에 비례해서 학습시간 소요
- Preview 통한 평가
	- 가만히 있을 경우 클래스 한 쪽의 값으로 나온다
	- 클래스에 맞는 동작을 하면 인식되는 것을 확인할 수 있다.
- 모델 이식해보기
	- Export Model 클릭
	- Uploade my model 클릭
	- 제공된 링크를 통해 하고자하는 곳에서 실행
